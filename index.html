How to link bot

Method 1 – QR scan (classic)
After starting bot, QR appears in console. Open WhatsApp → Linked Devices → Link a Device → scan QR.

Method 2 – Pairing code (fast)
Bot terminal shows a 12‑digit code. Go to WhatsApp settings → Linked Devices → Link with phone number → enter code.

Linking security token: During linking, the bot may ask for a verification key — type MZAZIBOT to finalize admin access.
After linking: Type .menu in any chat to see full command list. Bot responds instantly if online.

Frequently asked questions

❓ How do I get the pairing code if I missed it?+

Restart the bot from console — a new QR or pairing code will appear. Also you can type .pair in an already linked WhatsApp to generate fresh code.

🔐 Do I need the MZAZIBOT code every time?+

Only once during initial admin verification. After linking, the session is stored and auto‑reconnects.

🌐 Can I host on free platforms like Render?+

Yes, MoneyHeist AI works on Render, Railway, and Oracle free tier. Use persistent disk to keep session.

📱 Can I link multiple WhatsApp numbers?+

Absolutely! The bot supports multi‑session; each number requires a separate linking step via console.
